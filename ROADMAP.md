# Roadmap

## Near term

- Add calculation tests for the core total-cost formula.
- Add CSV export.
- Add preset rows for common Japanese mobile plan comparison cases.
- Improve keyboard and screen-reader affordances.
- Add optional memo templates for campaign deadlines and caveats.

## Later

- Add richer import validation and preview before replacing saved estimates.
- Add offline-first polish with a small web app manifest.
- Add documentation examples for MNP, device-only purchases, and SIM-only plans.

## Non-goals

- No user accounts.
- No server-side storage.
- No analytics or tracking.
- No automatic recommendation of a carrier or plan.
