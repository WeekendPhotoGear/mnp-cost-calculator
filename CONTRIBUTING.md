# Contributing

Contributions are welcome, especially small improvements that make the tool more
accurate, accessible, and understandable.

## Good first contributions

- Improve English or Japanese copy.
- Add accessibility fixes.
- Improve mobile layout.
- Add comparison presets for common contract periods.
- Improve JSON import validation.
- Add tests for cost calculations.

## Development

No build step is required.

```bash
python3 -m http.server 8765
```

Open <http://127.0.0.1:8765/>.

## Pull request expectations

- Keep the app static and dependency-free unless discussed first.
- Do not add analytics, ads, trackers, or backend calls.
- Explain any calculation behavior changes.
- Include screenshots for UI changes when practical.
