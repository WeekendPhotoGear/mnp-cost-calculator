# Changelog

All notable project changes are documented here.

## v0.2.0 - 2026-07-01

### Added

- Added anonymous sample estimates so first-time users can try the comparison
  flow without entering private plan details.
- Added structured GitHub Issue Forms for bug reports and feature requests.
- Added a Japanese README overview for the primary project audience.
- Added README guidance for feedback and issue reports.

### Notes

- No backend, analytics, tracking, login, or external API calls were added.
- The saved estimate storage format is unchanged.

## v0.1.1 - 2026-06-16

### Added

- Added maintainer quality checks with GitHub Actions.
- Added focused calculation and import/export tests.
- Added repository maintenance files for contributing, security, roadmap, and
  code ownership.

## v0.1.0 - 2026-06-16

### Added

- Initial public release of the local-first MNP cost calculator.
- Added English and Japanese UI.
- Added browser-only saved estimates using `localStorage`.
- Added JSON/CSV export and JSON import.
