## Project description

I am the owner and core maintainer of MNP Cost Calculator, a small local-first
open-source browser tool for comparing Japanese mobile plan switching costs.

The project helps users compare MNP and mobile carrier offers by calculating
total cost and effective monthly cost across 12, 24, 36, or 48 months. It stores
estimates only in the browser via localStorage and does not use a server, login,
analytics, tracking, or external APIs.

## Why it matters

Japanese mobile plan switching is financially meaningful but often difficult to
compare. Real costs can depend on temporary campaign discounts, device payments,
initial fees, points, cashback, and trade-in assumptions. This project provides a
transparent open-source calculator that users can inspect, run locally, and use
without sending personal estimate data to a server.
