# Privacy

MNP Cost Calculator is intentionally local-first.

## What is stored

The app can store estimates in the current browser using `localStorage`.
Saved fields include plan names, comparison periods, cost inputs, and notes.

## What is not collected

This project does not use:

- Accounts or login
- A backend server
- Analytics or tracking scripts
- Advertising scripts
- External APIs
- Cookies

## Export and import

Users can export estimates as a JSON file. That file is created locally by the
browser. Anyone who receives the exported JSON can read its contents, so users
should treat exported files as personal data.

## Clearing data

Saved estimates can be removed from the app with the clear button. Browser site
data controls can also remove stored estimates.
